Bookmark scripts


Bookmark scripts are a small peace of javascript that can be run on any page. To install one just click and drag the link to your bookmark bar. To run a script, click on the bookmark.

Rainbow -Rainbow is a script that will change the background of any page.
Page Destroyer -Page Destroyer is a script the will alow you to blow up a webpage. Hit 'k' for hotkey help. (KNOW BUG: Sometimes hotkeys won't load. Fix, refresh page and run again.)
